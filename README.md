# TechTactics – Logical Framework Builder

## Theme
Theme 3 – Program Design / Logical Framework Approach

## Problem Statement
Many education-focused NGOs and institutions struggle to convert
their ideas into a structured and measurable program design.
Planning is often unorganized, consultant-driven, and lacks clarity
between outcomes, activities, stakeholders, and indicators.

## Solution
This project provides a step-by-step Logical Framework Builder that
guides users through a structured planning process:

1. Define the problem clearly  
2. Specify student outcomes  
3. Plan key activities  
4. Identify stakeholders  
5. Set measurable indicators  

## Key Features
- 5-step guided program design workflow
- Simple and clean user interface
- Logical Framework summary generation
- Beginner-friendly implementation

## Tech Stack
- Python
- Django
- HTML
- CSS

## How to Run the Project
1. Clone the repository  
2. Create virtual environment  
   `python -m venv venv`
3. Activate environment  
   `venv\Scripts\activate`
4. Install dependencies  
   `pip install -r requirements.txt`
5. Run migrations  
   `python manage.py migrate`
6. Start server  
   `python manage.py runserver`
7. Open browser  
   `http://127.0.0.1:8000/`

## Outcome
This system converts unstructured educational ideas into a
complete Logical Framework Summary, helping organizations
plan programs efficiently.
